Recent Research Projects

Environment reconstruction using passive backscatter tags

August 2025 - Present

Passive tags sense inter-tag channel phase and attenuation, which are perturbed by environmental reflectors like walls and humans. This project uses those perturbations to reconstruct surroundings in a cheap and privacy perserving fashion.

Energy-aware localization for passive backscatter tags

February 2024 - June 2025

ML-based unambiguous ranging between passive tags (median error ~6.9 cm up to 2 m) and localization under 2.5 cm with 12 anchors. A link-quality heuristic cuts energy use by up to 51% with minimal accuracy loss.

Benchmarking vision tasks on edge and cloud

February 2024 - December 2025

Deployment decisions for real-time vision pipelines (edge, cloud, or hybrid) are non-trivial. This work provides comprehensive benchmarks across key tasks (detection, segmentation, etc.) with state-of-the-art methods to help in deployment decision-making process.

Collaborative Backscatter in passive backscatter tags

January 2023 - July 2024

Extended backscatter range by having helper tags find near-optimal phase configurations in polynomial time. Achieved ~20% median gain with static helpers and 3–4× gain with active helpers.
